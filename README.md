# Job Application Portal

A simple Job Application Portal where users can apply for jobs, and admins can manage job applications. This project is built using PHP and MySQL, offering basic functionalities for job seekers and administrators.

## Features

### Admin Panel:
- View all submitted job applications.
- Edit and update job application details.
- Delete job applications.

### User Panel:
- View available job listings.
- Apply for jobs by submitting personal and professional details.

## Technologies Used:
- **Backend:** PHP
- **Database:** MySQL
- **Frontend:** HTML, CSS

## Prerequisites:
Before running this project, ensure you have:
- PHP installed (v7.0 or above).
- MySQL or any other equivalent database system.
- A web server (e.g., Apache) or a local PHP server (like XAMPP, WAMP, MAMP, etc.).
